# Contribuyendo al Generador de Contraseñas Seguras de Tecnoyfoto

¡Gracias por tu interés en contribuir al Generador de Contraseñas Seguras de Tecnoyfoto! A continuación, encontrarás las directrices que te ayudarán a involucrarte con el proyecto.

## Cómo Contribuir

### Informar de Problemas

Si encuentras un error o tienes una sugerencia, por favor verifica primero que no haya sido reportado o sugerido anteriormente. Si es un problema nuevo, abre una issue con una descripción clara y completa, incluyendo pasos reproducibles si es posible.

### Sugerencias de Mejoras

Las mejoras son siempre bienvenidas. Si tienes ideas para mejorar el generador, por favor, abre una issue para discutirlo antes de trabajar en cualquier código.

### Pull Requests

Sigue estos pasos para enviar tu pull request:

1. Haz fork del repositorio y crea tu rama desde `main`.
2. Si has añadido código que debe ser probado, incluye tests relevantes.
3. Asegúrate de que tu código cumple con las guías de estilo del proyecto.
4. Actualiza el README.md con los detalles del cambio si es necesario.
5. Envía la pull request con una descripción detallada de los cambios.

## Código de Conducta

Todos los participantes en el proyecto deben adherirse a nuestro Código de Conducta, por favor lee el archivo CODE_OF_CONDUCT.md.

## Proceso de Revisión de Pull Requests

Los mantenedores del proyecto revisarán todos los pull requests. Nos esforzamos por responder lo más rápido posible, pero el tiempo de respuesta puede variar dependiendo de la carga de trabajo.

Gracias por contribuir al Generador de Contraseñas Seguras de Tecnoyfoto!
